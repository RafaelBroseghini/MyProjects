# Hangman :video_game:
In this side project I implemented a basic Hangman object that has all methods for a fun quick round of Hangman.
> Command line prompts guide you through the game.

## Features :link:
* Colored command prompt.
* Difficulty settings.

## Usage :grin:
`python main.py`
